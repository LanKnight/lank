### 2026-06-25 安装与跨目录使用优化

- [x] **`lank install` 命令** — 一键安装到系统，自动将 Scripts 目录添加到 PATH
- [x] **`lank doctor` 命令** — 诊断 Python 环境、依赖、配置、安装方式、命令可用性
- [x] **`lank version` 命令** — 显示版本号
- [x] **`lank.cmd` 启动脚本优化** — 简化逻辑，从任何目录都能通过 `py -m lank` 运行
- [x] **修复 `pkg_resources` 弃用警告** — 改用 `importlib.metadata`
- [x] **`lank install` 自动添加 PATH** — 通过 `reg add` 将 Scripts 目录添加到用户 PATH（修复 `setx` 截断问题）
- [x] **支持环境变量 `OPENAI_API_KEY`** — 自动读取系统环境变量，无需手动配置 API Key
- [x] **支持环境变量 `OPENAI_API_BASE` / `OPENAI_MODEL`** — 可通过环境变量覆盖 API 地址和模型
- [x] **配置来源标记** — `lank set show` 和 `lank doctor` 显示配置项来源（环境变量/配置文件）
- [x] **帮助信息更新** — 加入 `install`、`doctor`、`version` 命令说明
